# Skills

This directory contains the reusable skills shipped inside the `superdeveloper` bundle.

## Included skills

- [`brainstorming/`](brainstorming/) - idea generation, design exploration, and visual companion workflows
- [`dispatching-parallel-agents/`](dispatching-parallel-agents/) - splitting work across multiple agents
- [`executing-plans/`](executing-plans/) - structured execution against an approved plan
- [`finishing-a-development-branch/`](finishing-a-development-branch/) - completion and handoff workflow
- [`receiving-code-review/`](receiving-code-review/) - responding to review feedback
- [`repo-onboarding/`](repo-onboarding/) - deep repository onboarding and context building
- [`requesting-code-review/`](requesting-code-review/) - preparing work for review
- [`subagent-driven-development/`](subagent-driven-development/) - multi-agent implementation and review loops
- [`superdeveloper/`](superdeveloper/) - central orchestration skill for the package
- [`systematic-debugging/`](systematic-debugging/) - disciplined debugging workflows
- [`test-driven-development/`](test-driven-development/) - TDD-oriented implementation guidance
- [`using-git-worktrees/`](using-git-worktrees/) - worktree-based git workflow guidance
- [`using-superdeveloper/`](using-superdeveloper/) - onboarding and routing into the wider bundle
- [`verification-before-completion/`](verification-before-completion/) - final verification workflows
- [`writing-plans/`](writing-plans/) - plan authoring workflows
- [`writing-skills/`](writing-skills/) - authoring and improving skills

## Navigation note

Each skill folder should expose a `SKILL.md` entry point. Supporting references, scripts, prompts, or examples live beside that file when needed.
