# MCP

This directory contains MCP references bundled with the `superdeveloper` package.

## Included bundles

- [`code-review-graph/`](code-review-graph/) - Graph-oriented MCP package for repository indexing, graph queries, and review workflows.
- [`vault/`](vault/) - Memory-oriented MCP plugin for persistent records, facts, links, and checkpoints.

## Why this folder exists

These bundles live inside the `superdeveloper` package as packaged references and supporting integrations. The repository-level MCP packages live separately under the root [`mcp/`](../../../mcp/) directory.

## Layout

```text
skills/superdeveloper/mcp/
|- code-review-graph/
|- vault/
`- README.md
```
